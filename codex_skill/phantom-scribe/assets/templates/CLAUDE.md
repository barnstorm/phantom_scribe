# Claude Code Setup (Template)

If you use Claude Code, this project can include local agents in `.claude/agents/`.

## Install Agents Into This Repo
- macOS/Linux: `./phantom_scribe/install.sh --claude`
- Windows: `.\phantom_scribe\install.ps1 -Claude`

## Start
- Invoke `story-orchestrator` with your story thumbnail to generate the chapter plan and beat mapping.

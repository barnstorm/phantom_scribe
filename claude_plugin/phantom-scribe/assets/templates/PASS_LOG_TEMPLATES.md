# PASS LOG TEMPLATES

Use separate logs per editing phase to track work, findings, and status by chapter.

## Character Honing Log (Template)
- **Chapter:** _Name/number_
- **Beats:** _#-#_
- **Status:** _Not started / In progress / Complete_
- **Findings:** _Character consistency, knowledge checks, contradictions_
- **Actions:** _What you changed or need to change_
- **Notes:** _Hand-off info, risks_

## Line Editing Log (Template)
- **Chapter:** _Name/number_
- **Beats:** _#-#_
- **Status:** _Not started / In progress / Complete_
- **Focus Areas:** _Voice, pacing, clarity, rhythm_
- **Changes Made:** _Bullets of substantive edits_
- **Open Items:** _Questions or follow-ups_

## Copy Editing Log (Template)
- **Chapter:** _Name/number_
- **Beats:** _#-#_
- **Status:** _Not started / In progress / Complete_
- **Checks:** _Typos, grammar, punctuation, formatting_
- **Edits:** _Brief list of fixes_
- **Notes:** _Style decisions or remaining issues_

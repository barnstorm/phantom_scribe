3. Pick the first ❌ Not Started chapter in that Act.
4. Write or edit the chapter following the Act notes.
5. Update the Act file and this index when you finish.

- Mark chapter status in the Act file: `**Status:** ✅ COMPLETE (YYYY-MM-DD)`
- Add word count if useful.
- Update "Completed" and "Next Up" here.
- When an Act is done, flip its status in the table.
